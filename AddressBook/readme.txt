Возможности
- на сайте существует регистрация пользователей;
- гости и все пользователи могут просматривать все контакты;
- зарегистрированные пользователи могут создавать, редактировать и удалять только свои контакты;
- пользователь с правами администратора может редактировать/удалять любые контакты;
- над каждой таблицей с контактами реализована сортировка средствами ajax;
- для зарегистрированных пользователей существует страница "Мои контакты";
- на главной странице выводятся блоки с последними добавленными контактами и последними зарегистрированными пользователями;
- существует возможность поиска по первому символу средствами ajax;
- сайт работает на русском и английском языках. Есть система автоопределения языка по выбранному языку браузера. Возможно ручное переключение языка;
- при клике на контакт появляется всплывающее окно с деталями.
